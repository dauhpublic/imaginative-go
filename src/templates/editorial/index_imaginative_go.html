A free code sample in the form of Go web application

We believe that when learning programming language is by directly implement the knowledge into a project. This imaginative project will show doing web application in Go language. We don't use any framework and forgive us if the code still not efficient or optimal, since this sample just will give you an idea how to achieve something in Go language.

Why Imaginative Go?

Ideas about achieve something with Go lang

We hope that is project will give you some ideas about how to achieve something with Go programming language by looking into the code inside this imaginative web application project.

Plenty of working code samples

Actually now still small amount of sample included hahaha, but don't worry we will add it over time. Just sit back and relax, drink your coffee.

Easy to run (only with Docker)

We design it to be simple. You only need Docker to run this application. Even you don't need Go installed on your system. But you can also run with your local setup if you want.

You can contribute

Any contribution are appreciated. Even you can help us with your request, code or ideas. Just do pull request on our GitHub repository.
